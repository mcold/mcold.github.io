Методы словаря

clear

Очищает словарь

get

Возвращает значение по ключу Не вызывает ошибку даже если ключ отсутствует

d.get(<key>[, <default>])

>>> d = {'a': 10, 'b': 20, 'c': 30}

>>> print(d.get('b'))
20
>>> print(d.get('z'))
None

items

Возвращает list кортежей в виде пар key-value

update

Обновляет/Megre-ит значение по записи либо по словарю

>>> d1 = {'a': 10, 'b': 20, 'c': 30}
>>> d1.update(b=200, d=400)
>>> d1
{'a': 10, 'c': 30, 'b': 200, 'd': 400}
>>> d2 = {'a': 10, 'b': 20, 'c': 30}
>>> d1.update(d2)
>>> d1
{'a': 10, 'c': 30, 'b': 20, 'd': 400}

pop

Удалет ключ и возвращает значение При отсутствии ключа падает ошибка

popitem

Рандомно выбирает ключ и выполняет pop
