Staticmethod

Использование статического метода

Статический метод (обёрнутый декоратором staticmethod) в принципе соответствует статическим методам в C++ или Java Классический способ использования статических методов

class MyClass():

    @staticmethod
    def pi():
        return 3.14

if __name__ == "__main__":
    print(MyClass.pi())
