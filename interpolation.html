Интерполяция

Суть интерполяции

Интерполяция в Perl подразумевает возможность вывода значения переменной в строке Она осуществляется при использовании двойных кавычек. Одинарные кавычки выводят простой текст

Reference & dereference

Особый случай интерполяции достигается при использовании приема создания ссылки и ее использования

use strict;

print "1+2 = @{[ 1 + 2]}\n";

Фактически данная конструкция включает 3 этапа: * создание анонимного списка * формирование ссылки на список * получение списка по ссылке

Аналогино работает для hash

print "${{key => 1+2}}{key}\n";

3

my %hash = %{{1 => 'one', 2 => 'two'}};
for (keys %hash){
    print "$_\n";
}

for (values %hash){
    print "$_\n";
}

2
one
two

Тоже самое касается простых переменных и методов

print "1+2 = ${\( 1+2 )}\n";
print "1+2 = ${\do{ 1+2 }}\n";

Пример простого скаляра

print "${\scalar localtime}\n";
