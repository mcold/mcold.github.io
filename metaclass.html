Metaclass

Py2 & Py3

Версии Python по разному регламентируют синтаксис метакласса

Python2:

from abc import ABCMeta, abstractmethod

class A(metaclass=ABCMeta):
    pass

Python3:

class A():
    __metaclass__ = ABCMeta
    pass
